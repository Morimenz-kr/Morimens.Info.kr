# 비밀계약 추천 정리

## 기준

- 캐릭터 전체 목록: `data/character_manifest.json`
- 사이트에 적용된 추천/대체 비밀계약: `data/character_settings.json`
- 주옵/부옵 출처:
  - [모든 각성체 미스터리 정리(상시캐편)](https://arca.live/b/forgettingeve/166746144)
  - [모든캐릭 미스터리 정리(한정캐편)](https://arca.live/b/forgettingeve/166581443)
- 원문에서 계약명은 이미지로만 들어간 구간이 많아서, 아래 주옵 표에서는 계약명을 임의 추정하지 않고 원문 텍스트로 확인되는 용도/메모만 적었다.
